## ✨ Sobre o Projeto

Este é o meu portfólio pessoal — criado para apresentar minha trajetória, projetos, habilidades e tecnologias de forma clara, criativa e totalmente focada na experiência do usuário.

A proposta foi criar um site **moderno, rápido e com personalidade**, trazendo:

- 🧩 Um design voltado para devs
- ⚡ Performance e SEO otimizados
- 🎨 UI minimalista e detalhista
- 🌙 Tema escuro para uma identidade visual forte
- 📊 Estatísticas do GitHub integradas
- 🧠 Conteúdos carregados de forma inteligente
- 🗺️ Suporte a múltiplos idiomas

---

## 🧠 Funcionalidades Principais

### 🔹 Projetos Destacados
Uma área totalmente dedicada para mostrar os principais projetos com descrição, stack e link.

### 🔹 Integração com GitHub
Mostra estatísticas reais:
- Total de estrelas ⭐
- Repositórios favoritos
- Linguagens mais usadas
- Contribuições anuais

### 🔹 Animações Suaves
Transições fluidas graças ao Framer Motion / CSS animations.

### 🔹 Responsividade Total
Funciona perfeitamente no celular, tablet e desktop.
